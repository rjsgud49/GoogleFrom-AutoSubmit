🚀 Google Form 자동 제출 프로그램

Google Form 실습실 대여 신청을 자동으로 입력 및 제출하는 Python + Selenium 기반 자동화 프로그램입니다.
지정된 날짜, 인원, 체크박스, 텍스트 입력을 자동으로 수행하며,
로그 파일로 실행 결과를 기록합니다.

📌 기능 소개
✅ 1. Google Form 자동 입력

"대여 신청" 라디오 버튼 자동 클릭

다음 페이지 이동

날짜 자동 입력

실습실 선택(라디오 버튼)

인원 목록 자동 입력

체크박스(야자 등) 자동 클릭

대여 목적 자동 입력

🚀 2. 자동 제출

폼의 모든 값을 자동으로 채운 뒤 “제출”을 클릭합니다.

📝 3. 실행 로그 기록

submission_log.txt 파일에 다음 정보가 기록됩니다.

실행 시간

각 단계 성공/실패 여부

입력한 데이터 내용

오류 내용

👀 4. Headless 모드 지원

브라우저를 띄우지 않고 백그라운드에서 실행 가능
(옵션 설정으로 ON/OFF 가능)
